📜 How to Play with Friends

🌐 Setting Up (Online)

1
Create a Room

One person taps "Create Room", enters their name, and picks the number of rounds. They'll get a Room Code (like ABC12).

2
Share the Code

Share the Room Code with your friends via WhatsApp, message, or just tell them! Everyone opens the game link on their own phone/laptop.

3
Friends Join

Friends tap "Join Room", enter the code and their name. You need 4 to 10 players to play!

4
Host Starts the Game

Once everyone has joined, the host taps "Start Game" and the fun begins!

🎲 How Each Round Works

A
Get Your Secret Role

Each player gets a secret role on their own screen — only you see it! Roles include Raja, Rani, Minister, Doctor, Teacher, Gardener, Milkman, Farmer, Police, and Thirudan.

B
Police Reveals & Guesses

The Police is revealed to everyone. The Police must figure out who the Thief (Thirudan) is and pick them!

C
Results & Scores

All roles are revealed! Everyone sees who was who and the points earned.

💰 Points System

👑 Raja (King) — 1000 pts
💎 Rani (Queen) — 500 pts
📜 Minister — 400 pts
👨‍⚕️ Doctor — 350 pts
👨‍🏫 Teacher — 300 pts
🌺 Gardener — 250 pts
🥛 Milkman — 200 pts
👨‍🌾 Farmer — 200 pts
🛡️ Police — 800 pts if catches thief, 0 if wrong
🦹 Thief — 0 pts if caught, 800 pts if escapes!

🏆 Winning

After all rounds, the player with the highest total score wins! The key is luck (getting high-point roles) and skill (the Police must observe and guess correctly, the Thief must act natural!).

💡 Tips for Playing with Friends

  • 🎤 Use a voice/video call while playing — it's way more fun when you can see reactions!
  • 🤫 Keep a poker face — if you're the Thief, don't give it away!
  • 🕵️ Police: Watch reactions — when roles are revealed, see who looks nervous!
  • 😈 Thief: Act confident — accuse someone else of being the thief to throw off the Police!
  • 📱 Everyone needs their own device — phone, tablet, or laptop all work!
  • 🌐 Same Wi-Fi network — all players must be on the same network, or deploy the game online!
